# 전투 시스템 명세

## 핵심 리소스
- 색상 스택: 빨강(피), 파랑(영혼), ...
- 생성: 플레이어 카드
- 소비: 동료 행동

## 턴 구조
1) 플레이어 카드 단계
2) 선공 판정 단계 (매 턴)
3) 선공 측 행동 단계
4) 후공 측 행동 단계
5) 종료/상태 갱신

## 선공 판정 규칙 (확정)
매 턴 시작 시 아래 값을 비교해 선공을 결정한다.

- 아군 선공 점수 = (아군 동료가 보유한 스택 합) + (이번 턴 플레이어가 사용한 카드의 스택 합)
- 적 선공 점수 = 몬스터가 보유한 스택 합

판정:
- 아군 선공 점수 > 적 선공 점수 → 아군 선공
- 아군 선공 점수 < 적 선공 점수 → 적 선공
- 동점일 때 → 랜덤으로 선공 결정

## 동점 연출 규칙 (확정)
- 동점 발생 시 시스템적으로는 랜덤 선공을 적용한다.
- 게임 내 UI/연출은 **코인 토스(앞/뒤)** 방식으로 선공 결과를 보여준다.
- 레퍼런스: **유희왕 마스터 듀얼의 코인 토스 연출**
