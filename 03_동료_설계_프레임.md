# 동료 설계 프레임

## 기본 원칙
- 역할군은 딜러 / 탱커 / 서포터 구조를 기본으로 하며 중복 편성을 허용한다.
- 동료는 전투 전에 배치하고 전투 중에는 교체/배치 변경이 불가능하다.
- 최대 배치 제한은 4명(4인 고정)이다.
- MVP 기준 동료 풀은 6종이며, 이 중 전투에는 4명을 편성한다.

## 행동 규칙
- 동료는 공용 스택을 바탕으로 자동으로 스킬을 사용한다.
- 자동행동은 전투 전 배치한 순서대로 동작한다.
- 스택이 부족하면 행동을 스킵한다.
- 행동 스킵 자체에는 별도 페널티를 두지 않는다.

## 모집 / 성장
- 동료는 전투로 획득한 재화(영혼석)를 소모해 모집 노드에서 모집한다.
- 재화가 없으면 모집 노드에 도달해도 모집할 수 없다.
- 모집한 동료는 조합하여 상위 동료로 승급하는 구조를 추후 도입한다.

## 동료 카드 구성 요소
- 이름
- 역할군(딜러/탱커/서포터)
- 직업군(캐스터, 오펜서, 디펜더, 어택커, 프리스트, 샤먼)
- 성향
- 요구 스택(속성/수치)
- 패시브 효과
- 스킬 1
- 스킬 2
- 행동 효과
- 모집 비용
- 시너지 대상

> 아래 성향/직업 시너지 항목은 **추후 확장 초안**이며, MVP 구현 기준에서는 후순위로 둔다.
> 패시브 관련 메타 확장은 프로토타입 이후 우선 검토 대상으로 둔다.

## 성향 설계안
- 한탕주의자: `-5` 혹은 `+5`만 사용
- 안전제일: `-1 ~ +3`
- 기회주의자: `-3 ~ +4`
- 낙천주의: `-5 ~ +5`

## 성향 시너지 초안
- 동일 성향 2인 배치시 발동
- 추가 배치 1인마다 % 증가량 추가
- 한탕주의자: 전투 후 획득 골드 `±N%`
- 안전제일: 전투 후 획득 골드 `+N%`
- 기회주의자: 전투 종료 후 남은 체력 비례 골드 `+N%`
- 낙천주의: 전투 후 스트레스 감소 `N%`

## 직업 설계안 (추후)
- 딜러 
	- 캐스터, 오펜서 
- 탱커 
	- 디펜더, 어택커
- 서포터
	- 프리스트, 샤먼

## 직업 시너지 초안
동일 직업 2인 배치시 발동
추가 배치 1인마다 % 증가량 추가
-  딜러  
	- 오펜서 : 단일 딜 증가 
	- 캐스터 : 범위 딜 증가
- 탱커
	- 디펜더 : 전체 체력 증가(실드 느낌으로 별도 그래픽 처리)
	- 어택커  : 본인 체력 증가
- 서포터
	- 샤먼 : 적 받는 피해량 증가
	- 프리스트 : 아군 받는 피해량 감소

## 용병소 운영 규칙 (2026-03-10 반영)
- 교체/편성 변경은 **용병소 노드에서만** 가능하다.
- 모집한 동료는 먼저 **장바구니(임시 덱)** 에 들어간다.
- 장바구니/임시 변경사항은 용병소 노드 내부에서만 유지되며, 노드를 벗어나면 리셋된다.
- 대기석 기본 슬롯은 **9칸**으로 시작한다.

## 합성 규칙 (확정)
- 같은 역할군 + 같은 성급 동료 3개를 합성하면,
- 한 단계 상위 성급의 **랜덤 역할군 동료**를 획득한다.
