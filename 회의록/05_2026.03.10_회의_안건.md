### A. 용병소 규칙 정하기
- [x] 교체 타이밍(노드 진입/즉시/전투 전)

- [x] 모집 방식(직접 선택/자동)

  - 모집 시 동료 카드는 먼저 **장바구니(임시 덱)** 로 들어감
- [x] 비용(영혼석) 수치
  - 모집 비용은 동료 성급에 따라 차등
  - 리롤/모집 모두 비용 발생, 패시브로 비용 감소 가능
- [x] 교체 후 유지/초기화 항목 정의
  - 장바구니/임시 변경사항은 **용병소 노드 내부에서만 유지**
  - 노드를 벗어나면 리셋되므로, 이탈 전 등록/합성 필요
- [x] 용병소 패시브
  - 패시브 레벨업으로 모집 카드 수 증가 또는 등장 성급 상향
  - 대기석 기본 슬롯: **9칸**
- [x] 합성 규칙
  - 같은 역할군 + 같은 성급 동료 3개를 합성해
  - 한 단계 상위 성급의 **랜덤 역할군 동료** 획득
