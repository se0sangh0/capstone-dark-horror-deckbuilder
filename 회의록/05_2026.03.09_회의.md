#  캡스톤

- 목표: 핵심 규칙 확정 + MVP 범위 잠금

## 안건 (우선순위 순)

### 1) 카드 규칙 확정 (최우선)
- 카드 속성 범위: 색상 / 숫자 / 키워드 어디까지 채택? 
	- 할리갈리 방식 (선택)
- 핸드 규칙: 매 턴 리셋 vs 일부 이월
	- 손패 장수 매턴 리셋(4장 사용했을 경우 4장 다시 드로우)
	- 손패 장수 리미트  (동료가 늘수록 +1) 
- 턴당 카드 사용 장수 / 비용 유무
	- 최소 사용 제한 없음
	- 최대 사용 제한 없음 
	- 더 이상 드로우 할 카드가 없는 경우 번뎀(하스스톤 탈진 시스템 레퍼런스)
- 덱 매수 제한 
	- 동료의 수에 따라서 증가
	- 동료 당 10장씩 추가됨
	
### 2) 공용 스택 시스템 상세
- 스택 상한 존재 여부
	- 상한 없이
- 잉여 스택 처리: 유지 / 소멸 / 오염 전환
	- 전투 중 자동으로 소모
- 스택 부족 시 처리: 무조건 스킵 vs 대체 행동
	- 무조건 행동 스킵

### 3) 동료 행동/배치 규칙
- 배치 순서 고정 여부, 전투 중 변경 가능 여부
	- 전투 전에 배치하고 들어가야하고 전투 중 배치 변경 불가
- 동료 행동 단위 
	- 스택을 바탕으로 해당하는 스킬을 자동으로 동작
	- 우선 순위는 배치한 순서대로 동작
- 스킵 페널티(스트레스/디버프/체력 손실)
	- 없음
- 동료별 역할
	- 딜탱힐 구조
	- 역할 중복 가능 (전열 4인 / 후열 4인 등)
- 동료 모집 방법
	- 전투로 획득한 재화를 소모해서 동료 모집 노드에서 모집 가능
		- 재화가 없으면 모집 노드에 도달해도 모집 불가
	- 최대 배치 제한 4명 (4인 고정)
	- 모집한 동료를 조합하여 상위 동료 획득(예정)


### 4) 스트레스/호러 페널티 설계
- 스트레스 범위(예: 0~100)
	- 다키스트 던전 패닉 시스템
		- 동료 타입 별로 디버프 발생
- 회복 수단(이벤트/카드/성소)
	- 노드 방문
- 스트레스 획득 조건
	- 피격 (입은 데미지 비례)
	- 동료 사망

### 5) 맵 노드 밸런스 초안
- 노드 구성
	- 히든 노드(? 노드) : 
		- 교회 : (재화소모) 사망한 동료 부활 || 힐, (재화소모x)스테레스 관리
		- 용병소 : (재화소모) 동료 모집
		- 약탈 노드 : (재화소모) 지나가기, 엘리트몹과 전투 돌입하기
	- 오픈 노드 : 
		- 전투 노드 : 적 전투
		- 화툿불 노드 : 보스 진입 전 재정비 노드 전체 체력 퍼센트로 회복 || 랜덤 회복
		- 보스 노드 : 보스 진입
	
### 6) MVP 스코프 잠금
- MVP 최소 기능 수치화
  - 카드 3종 * 스택-5~5 범위의 카드 
  - 동료 메인 직업군 이미지 기반 남여로 6종 생성
  - 적 
	  - 노말몹 3종
	  - 엘리트 2종
	  - 보스 1종
  - 노드 4종

### 7) 재화 구성
- 영혼석 : 노드 내 사용 재화
	- 동료 모집
	- 동료 부활
	- 회복
	
- 마석 : 마을에서 사용할 재화
	- 패시브 해금
	 
### 8) 동료
- 모집하는 동료별로 패시브 효과 및 스킬 구성이 다름
- 카드 구성은 역할군, 스킬1, 스킬2
	- 역할군 : 딜 탱 힐
	- 스킬1,2 : 직업군 별로 테이블 만들어서 랜덤 부여 
	- 패시브 스탯 (스플랜더 개발카드 레퍼런스 보유 효과로 스탯 보유량 보조 효과)
- 동료 성향 기재
	- 한탕주의자 : -5 혹은 +5로만 구성
	- 안전제일 : -1~+3사이로만 구성
	- 기회주의자 : -3~+4 사이로 구성
	- 낙천주의 : -5~에서+5사이로 구성
- 시너지(동일 성향으로 4인 구성시에 붙는 보너스)
	- 한탕주의자 : 전투 후 획득하는 골드 +-N%
	- 안전제일 : 전투 후 획득하는 골드 +N%
	- 기회주의자 : 전투 종료 후 남은 체력 비례 골드 +N%
	- 낙천주의 : 전투 후 스트레스 감소 N%
### 9) 역할 분담 + 일정
- 시스템/콘텐츠/프로토타입/문서 담당자 지정
- 다음 회의 전 개인 과제 1개씩 확정

## 회의 결과 요약 (회의 후 작성)
- 핵심 결정:
  - 카드 표현은 할리갈리식의 단순한 속성/수치 중심 구조로 간다.
  - 손패는 매 턴 리셋하며, 사용 장수만큼 다시 드로우한다.
  - 공용 스택은 상한 없이 누적되고 부족하면 해당 동료 행동은 스킵한다.
  - 동료는 전투 전에만 배치하고, 딜/탱/힐 역할군 구조를 사용한다.
  - MVP 범위는 카드 3종, 동료 6종, 적 3/2/1, 노드 4종으로 잠근다.
- 미결 이슈:
  - 시작 손패 기본 장수와 시작 덱 기본 매수
  - 노드 4종 우선 구현 조합
  - 스트레스 구간별 수치와 타입별 디버프 상세값
  - 역할 분담 및 개인 과제 배정
- 액션 아이템:
  - [ ] 서상호: 손패 / 스택 / 자동행동 수치 초안 작성 및 문서 취합
  - [ ] 권성현: 직업군 스킬 템플릿 보강 / 성향 / 시너지 표 작성
  - [ ] 고성모: 적 3 / 2 / 1 구성과 노드 4종 골격 구현
  - [ ] 전원: 발표용 규칙 요약본과 테스트 체크리스트 검수
