# 괴이탐사국 UI 프로토타입 (Figma Make)

> 다크 호러 덱빌딩 오토배틀러 게임 UI 프로토타입
> Figma Make를 사용한 React + Tailwind CSS 구현

## 프로젝트 개요

이 폴더는 Figma Make를 사용하여 제작한 **괴이탐사국** 게임의 UI 프로토타입입니다.
기획서의 디자인 가이드라인을 기반으로 실제 게임 화면을 구현했습니다.

### 구현된 기능

#### 1. 노드 시스템
- ✅ 10층 구조 (층당 3갈래)
- ✅ 노드 타입: 전투(60%), 용병소(40%), 화툿불(9층 고정), 보스(10층 고정)
- ✅ 노드별 아이콘 및 색상 구분

#### 2. 전투 시스템
- ✅ 턴 페이즈 시스템 (카드 플레이 → 선공 판정 → 전투 → 결과)
- ✅ 선공 판정: 스킬 코스트 기반
  - 캐스터(4) + 오펜더(7) + 디펜더(7) + 어택커(5) + 프리스트(8)
- ✅ 덱 시스템: 40장 덱 (파티원 4명 × 10장/명)
- ✅ 손패: 배치 동료 수 (4장)
- ✅ 드로우: 턴 종료 시 사용한 카드 수만큼 재드로우

#### 3. 카드 시스템
- ✅ 성향별 스택 생성 (0 제외)
  - 도박사: -5 또는 +5 (50:50)
  - 안전주의자: [-1, 1, 2, 3]
  - 기회주의자: [-3, -2, -1, 1, 2, 3, 4]
  - 낙천가: [-5, -4, -3, -2, -1, 1, 2, 3, 4, 5]
- ✅ 역할별 카드: 딜러, 탱커, 서포터

#### 4. 동료 시스템
- ✅ 용병소에서 동료 모집
- ✅ 판타지 이름 자동 생성 (Aldric Ashveil 등)
- ✅ 직업: 캐스터, 오펜더, 디펜더, 어택커, 프리스트, 샤먼(추후)
- ✅ 파티 편성: 4인 고정
- ✅ 전열/후열 배치

#### 5. 화툿불 시스템
- ✅ 9층 고정 배치 (보스 직전 휴게소)
- ✅ 체력 +30, 스트레스 -15 회복

#### 6. UI/UX
- ✅ 다크 호러 톤앤매너 적용
- ✅ 고정 UI (좌측 패널) + 중앙 콘텐츠 전환
- ✅ Unity 좌표 주석 포함 (1920x1080 기준)
- ✅ 황동, 청록, 보라, 붉은색 강조색 사용

## 기술 스택

- **React** 18.3.1
- **Tailwind CSS** 4.1.12
- **Vite** 6.3.5
- **TypeScript**
- **Radix UI** (UI 컴포넌트)
- **Lucide React** (아이콘)

## 기획서 기준 구현 상태

### ✅ 완료
1. 노드 시스템 (10층, 3갈래, 화툿불 9층 고정)
2. 전투 턴 구조 (드로우 → 카드 → 선공 판정 → 전투)
3. 선공 판정 (스킬 코스트 기반)
4. 카드 시스템 (40장 덱, 성향별 스택)
5. 동료 시스템 (이름 생성, 모집)

### ⏳ 백로그
1. 동료 사망 처리
2. 스트레스 시스템 (패닉 판정)
3. 실제 스킬 발동 로직
4. 적 AI
5. 교회 노드
6. 엘리트 전투 노드

## 기획서 수정 제안

구현 과정에서 발견한 기획서 수정 제안사항:

1. **화툿불 노드**: 9층 고정 (보스 직전 휴게소)
2. **성향별 스택**: 0 제외 명시 필요
3. **선공 판정**: 스킬 코스트 기반으로 수정
4. **전투 시작**: 초기 손패 드로우 명시

자세한 내용은 메인 리포지토리의 기획서를 참조하세요.

## Unity 이관 시 주의사항

모든 컴포넌트에 Unity 좌표 주석이 포함되어 있습니다.
기준 해상도: **1920 x 1080**

```typescript
/* Unity 좌표:
 * X: 320px (좌측 패널 이후)
 * Y: 64px (상단 바 이후)
 * Width: 1600px
 * Height: 816px
 */
```
