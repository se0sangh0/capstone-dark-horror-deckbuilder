## 2026-03-09 회의 반영 사항
- 카드 표현 방식은 할리갈리식의 직관적인 속성/수치 중심으로 단순화한다.
- 손패는 매 턴 리셋하고, 사용한 장수만큼 다시 채우는 흐름을 채택한다.
- 공용 스택은 상한 없이 누적되며, 부족하면 해당 동료 행동은 스킵한다.
- 동료는 전투 전 배치 순서를 확정하고 전투 중에는 변경하지 않는다.
- 동료 역할은 딜/탱/힐 구조를 유지하되 역할 중복을 허용한다(전열 4인/후열 4인 구성 가능).
- 자동행동 우선순위는 코스트 우선안에서 배치 순서 우선으로 변경한다.
- 스트레스는 피격 피해량과 동료 사망을 중심으로 누적되며, 노드 방문으로 관리한다.
- MVP 범위는 카드 3종, 동료 6종, 적 6종(3/2/1), 노드 4종으로 잠근다.
- 팀은 3인 구조로 진행하며, 팀장 서상호 / 팀원 고성모 / 팀원 권성현 기준 운영 초안을 사용한다.
- 직업군 스킬 문서는 아직 합의 전이므로 템플릿 수준으로 유지한다.

## 2026-03-15 회의 반영 사항
- 자동행동 구조는 유지하되, MVP에서는 단순 우선순위 규칙을 먼저 구현한다.
- 미행동 보상 규칙을 채택한다.
  - 미행동 시 해당 역할 스택 +1
  - 다음 턴 행동 순서를 미행동 동료 우선으로 재정렬
- 동료 모집 시 스킬 랜덤 구조는 유지하되, 중복 방지 규칙을 둔다.
- 성향 시스템과 복잡한 시너지는 MVP 핵심축에서 후순위로 내린다.
- 이번 버전의 목표를 ‘돌아가는 플레이 가능 프로토타입’으로 재확인한다.
- 이번 주 핵심 산출물에 전투 UI 초안을 포함한다.
- UI 기준 해상도는 우선 FHD(1920x1080)로 잡는다.
- 최종 제출 마감은 교수님 일정 확인 후 재확정한다.
