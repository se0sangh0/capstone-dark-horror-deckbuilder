# UI 구성 규정 및 분리안 (발라트로식 고정 UI + 패널 전환)

## 문서 목적
- 씬 구성 방식에 대한 팀 공통 규칙을 확정한다.
- 전투/노드/용병소/교회 화면 전환 기준을 통일한다.
- 구현 시 상태 손실/중복 로직/충돌을 줄인다.

---

## 1) 최종 채택 아키텍처
**하이브리드 구조 채택**
- 고정 UI(좌측 상태 패널 + 공통 HUD)는 항상 유지
- 중앙 콘텐츠 영역만 패널 단위로 전환
- 전체 씬 전환은 최소화(런타임 부트/메인 메뉴 등 예외 상황만)

### 채택 이유
1. 상태 연속성 유지에 유리 (덱/스택/파티/스트레스)
2. 디버깅 단순화 (씬 경계 버그 감소)
3. UX 일관성 확보 (플레이어가 맥락을 잃지 않음)

---

## 2) 화면 구성 규정

### 2-1. 고정 UI 영역 (Persistent UI)
항상 표시:
- 재화: 영혼석, 마석
- 파티 상태: 4인 슬롯, 역할, 성급, 핵심 스탯 요약
- 덱/스택 요약: 덱 매수, 현재 스택, 턴/라운드 요약
- 스트레스 상태: 현재값, 구간(안정/압박/패닉)
- 공통 버튼: 설정, 로그, 도움말

### 2-2. 중앙 콘텐츠 패널 (Swappable Panel)
패널 전환 대상:
- 전투 패널
- 노드 이동 패널
- 용병소 패널
- 교회 패널

규칙:
- 동시에 하나의 메인 패널만 활성화
- 패널 진입/이탈 시 `OnEnter/OnExit` 훅 강제
- 상태 저장은 전역 상태 저장소(GameState)에서만 수행

### 2-3. 팝업 사용 기준
팝업으로 처리:
- 짧은 의사결정(보상 선택, 확인/취소)
- 상세 정보 조회(툴팁, 카드 상세)
- 경고/검증(재화 부족, 덱 제한 초과)

팝업 금지:
- 장시간 체류/복합 작업(전투 본체, 용병소 메인 작업)

---

## 3) 상태 관리 규정

### 3-1. 단일 진실 소스(Single Source of Truth)
아래 상태는 반드시 `GameState`에서 관리:
- 파티 편성
- 덱/손패/버림패
- 공용 스택
- 스트레스
- 재화(영혼석/마석)
- 노드 진행도

패널은 상태를 “소유”하지 않고, 조회/요청만 한다.

### 3-2. 상태 변경 규칙
- 패널 내부 임시값은 `DraftState`로 관리
- 확정 액션(예: 용병소 적용 버튼)에서만 `GameState` 반영
- 취소 시 Draft 폐기

### 3-3. 로그 규칙
중요 변경은 Action Log 기록:
- 동료 모집/교체/합성
- 재화 증감
- 스트레스 급변
- 전투 결과 반영

---

## 4) 패널별 분리안

### A. 전투 패널
책임:
- 턴 진행
- 자동행동(배치 순서 우선)
- 스택 소비/행동 스킵 처리
- 전투 결과 산출

입력:
- 파티/덱/스택/적 정보(GameState)
출력:
- 전투 결과(HP/스트레스/보상 변경)

### B. 노드 이동 패널
책임:
- 현재 노드 표시
- 다음 노드 선택
- 노드 진입 트리거 실행

입력:
- 맵 진행도, 현재 위치
출력:
- 진입 노드 타입 확정

### C. 용병소 패널
책임:
- 모집/리롤
- 장바구니(임시 덱) 관리
- 교체/합성 처리

핵심 규칙(확정):
- 용병소에서만 교체 가능
- 장바구니 변경은 노드 내부에서만 유효
- 노드 이탈 시 미확정 변경 리셋
- 대기석 기본 9칸

### D. 교회 패널
책임:
- 스트레스 관리
- 회복/정화 선택
- 비용/효과 적용

---

## 5) 구현 우선순위 (MVP)
1. 루트 씬 + 고정 UI 프레임 완성
2. 패널 라우터(전투/노드/용병소/교회 전환) 구현
3. GameState + DraftState 분리
4. 용병소/전투 핵심 루프 연결
5. 팝업 시스템(확인/보상/경고) 추가

## 7) 팀 협업 규정
- UI 구조 변경 PR은 반드시 이 문서 업데이트와 함께 제출
- 패널 책임 변경 시 `책임/입출력` 섹션 동시 수정
- 회의에서 구조 변경 확정 시 허브 문서 링크 동기화

## 8) 검증 체크리스트
- [ ] 고정 UI가 패널 전환 후에도 유지된다.
- [ ] 전투/노드/용병소/교회가 동일 루트에서 전환된다.
- [ ] 패널 이탈 후 상태 유실이 없다.
- [ ] 용병소 임시 변경(장바구니)이 이탈 시 리셋된다.
- [ ] 팝업이 메인 패널 책임을 침범하지 않는다.
