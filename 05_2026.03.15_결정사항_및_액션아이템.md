# 2026.03.15 결정사항 / 미결정 / 액션아이템

## 1) 결정된 것

### 전투 / 시스템
- 자동행동 구조는 유지한다.
- 미행동 보상 규칙을 채택한다.
  - 미행동 시 해당 역할 스택 +1
  - 다음 턴 행동 순서를 미행동 동료 우선으로 재정렬
- 초기 구현은 복잡한 발동 규칙보다 단순 우선순위 규칙을 우선한다.
- 카드 → 공용 스택 → 동료 행동의 기본 흐름은 유지한다.

### 동료 / 스킬
- 동료 모집 시 스킬 랜덤 부여 구조를 유지한다.
- 중복 스킬이 과하게 겹치지 않도록 중복 방지 규칙을 둔다.

### MVP 범위
- 이번 버전의 목표는 ‘플레이 가능한 프로토타입’이다.
- 우선 구현 범위:
  - 맵 노드
  - 전투
  - 플레이어 화면 / UI
  - 기본 보스 흐름
  - 기초 시스템
- 콘텐츠 확장보다 시스템 완성도를 우선한다.

### UI 방향
- 이번 주 핵심 산출물 중 하나는 UI 초안이다.
- 기준 해상도는 우선 FHD(1920x1080)로 잡는다.
- 해상도 대응/디테일 폴리싱은 후순위다.

## 2) 아직 미결정인 것
- 자동행동의 세부 우선순위 규칙 최종안
- 시너지를 MVP에 완전히 제외할지, 대표 사례 1개만 넣을지
- 성향 시스템을 문서상으로만 남길지, 최소 기능으로 유지할지
- 제출/중간 점검의 최종 마감일
- 교수님 일정 확인 후의 실제 주차별 마감 기준
