# 코어루프 명세 (2026-03-15 확정)

## 전체 루프
1. 노드 선택 → 2. 동료 배치 → 3. 턴 시작 (손패 리셋) → 4. 카드 플레이 → 5. 자동 전투 → 6. 결과 처리 → 7. 노드 보상 → 8. 다음 노드

**핵심**: 전투 전 배치 → 공용 스택 생성 → 자동 행동 → 결과

## 확정 규칙

### 손패
- 매 턴 리셋, 사용 장수만큼 재드로우
- 상한 없음 (동료 수 증가 시 손패 상한 증가 적용)
- 탈진 시 번데미지

### 공용 스택
- 상한 없음, 턴 종료 시 자동 소모
- 부족 시 동료 행동 스킵 (대체 행동 없음)

### 동료 행동
- 전투 전 배치 순서 확정 (전투 중 변경 불가)
- **우선순위**: 배치 순서 → 미행동자 우선 재정렬
- 미행동 보상: 해당 스택 +1, 다음 턴 맨 앞 배치
- 예시: `1-2-3-4` → 3 미행동 → 다음 턴 `3-1-2-4`

### 재화
| 재화 | 용도 | 위치 |
|------|------|------|
| 영혼석 | 모집/부활/회복 | 탐험 노드 |
| 마석 | 메타 성장 | 백로그 |

## MVP 범위
- 노드: 오픈(전투/화툿불/보스) + 이벤트(용병소/교회⏳/엘리트전투⏳) 총 6종
- 동료: 5종 (MVP) + 1종 백로그 (샤먼) / 파티 4인 고정, 모집으로 교체
- 카드: 3종 (딜러/탱커/서포터)
