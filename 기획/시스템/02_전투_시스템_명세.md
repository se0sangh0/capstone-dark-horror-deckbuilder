# 전투 시스템 명세 (2026-03-15 확정)

**참조**: [코어루프 명세](../코어/01_코어루프_명세.md)

## 턴 구조
1. 턴 시작 (손패 리셋, 코어루프 참조)
2. 플레이어 카드 플레이
3. 선공 판정
4. 선공 측 행동
5. 후공 측 행동
6. 결과 처리 (스트레스/사망)

**MVP**: 동료/적 턴당 1회 행동

## 선공 판정

**아군 점수** = Σ(동료 스택) + 이번 턴 카드 스택 합  
**적 점수** = Σ(적 스택)

| 비교 | 결과 |
|------|------|
| 아군 > 적 | 아군 선공 |
| 아군 < 적 | 적 선공 |
| 동점 | 코인 토스 (랜덤) |

**연출**: 유희왕 코인 토스 참조

## 스트레스
| 원인 | 증가 |
|------|------|
| 피격 | 피해량 비례 |
| 동료 사망 | 고정량 |
| 누적 | 타입별 디버프 (스트레스 표 참조) |

**참조**: [스트레스 디버프 표](02_스트레스_디버프_표.md)

## 동료 역할
- 기본: 딜러/탱커/서포터
- **중복 허용** (예: 딜러 3 + 서포터 1)
- 파티: **4인 고정** (모집으로 교체, 코어루프 참조)

**자동행동 우선순위**: 코어루프 명세 참조
