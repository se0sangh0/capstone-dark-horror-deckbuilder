# 맵 노드 설계

## 전체 노드 구조 (2026-03-09 반영)

### 히든 노드 (`?`)
- 교회
  - 재화 소모: 사망한 동료 부활 또는 체력 회복
  - 무료 기능: 스트레스 관리
- 용병소
  - 재화 소모: 동료 모집
- 약탈 노드 (엘리트몹)
  - 재화 소모 후 안전하게 지나가기
  - 또는 엘리트 전투 강제 진입

### 오픈 노드
- 전투 노드
  - 일반 적과 전투
- 화톳불 노드
  - 보스 전 재정비
  - 전체 체력 퍼센트 회복 또는 랜덤 회복
- 보스 노드
  - 보스 전투 진입

## 설계 메모
- 전체 설계 노드는 히든/오픈을 합쳐 더 많지만, MVP 구현은 4종으로 제한한다.
- 어떤 4종을 우선 구현할지는 노드 가치와 구현 난이도 기준으로 다음 결정이 필요하다.
