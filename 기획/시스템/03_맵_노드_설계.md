# 맵 노드 설계

## 확정 구조 (2026-04-08 확정)

| 항목 | 값 | 비고 |
|---|---|---|
| 총 층 수 | 10층 | 전속 테스트 후 조정 가능 |
| 층당 갈래 | 3갈래 | 좌/중/우 노드 선택 |
| 최종층 | 10층 | 보스 노드 고정 |

## 노드 생성 방식
- **자동 생성 알고리즘** 기반으로 자동 구현
- 노드 배치 알고리즘 참조 디자인: [Claude 손드 노드 생성 알고리즘](https://claude.ai/share/903db421-5150-4490-bc9b-e87a2359fcee)
- 전투 속도 테스트 후 층 수 조정 가능

## 노드 배치 규칙
- 마지막(10층)은 보스 노드로 고정
- MVP 4종 노드가 충분한 비율로 배분되도록 알고리즘에서 가중 설정
- 일반 전투 노드 비율이 가장 높고, 용병소 → 교회 → 비고 타입 순으로 다음

## 전체 노드 구조 (2026-03-09 반영)

### 히든 노드 (`?`)
- 교회
  - 재화 소모: 사망한 동료 부활 또는 체력 회복
  - 무료 기능: 스트레스 관리
- 용병소
  - 재화 소모: 동료 모집
- 약탈 노드 (엘리트랍)
  - 재화 소모 후 안전하게 지나가기
  - 또는 엘리트 전투 강제 진입

### 오픈 노드
- 전투 노드
  - 일반 적과 전투
- 화톯불 노드
  - 보스 전 재정비
  - 전체 체력 퍼센트 회복 또는 랜덤 회복
- 보스 노드
  - 보스 전투 진입

## 설계 메모
- 전체 설계 노드는 히든/오픈을 합쳐 더 많지만, MVP 구현은 4종으로 제한한다.
- 히든 노드 전체, 화톯불 노드 등은 MVP 제외(백로그) 대상이다.
