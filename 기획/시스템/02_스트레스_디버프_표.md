# 스트레스 디버프 표

## MVP 구현 원칙
- 초기 MVP는 구간 효과를 단순화하고, 패닉 분기는 2종(공포 경직/과호흡)만 우선 구현한다.

## 기본 범위
- 스트레스 범위: `0 ~ 100`
- 상승 조건
  - 피격 시 입은 피해량 비례 상승
  - 동료 사망 시 추가 상승
- 기본 회복
  - 교회 노드 무료 스트레스 관리
  - 휴식 계열 노드 또는 이벤트
  - 추후 카드/패시브 회복 수단 확장

## 구간별 상태

### 0 ~ 50: 안정
- 공통 효과 없음
- UI는 정상 상태로만 표기

### 51 ~ 99: 압박
- 공통 디버프
  - 피격 시 추가 스트레스 증가량 `+10%`
  - 스킬 퍼센트 포인트 감소 (`-N%`)

### 100: 패닉
- 진입 시 즉시 1회 패닉 판정
- 패닉 결과 예시
  - 공포 경직: 해당 턴 행동 불가
  - 과호흡: 스택 요구량 `+1`
- 패닉 이후 스트레스는 `50`으로 되돌리고, 해당 전투 종료 전까지 역할별 중증 디버프 유지

## 역할별 중증 디버프 (추후)

### 딜
- 목표: 화력은 유지하되 불안정성 증가
- 효과
  - 받는 피해량 증가

### 탱
- 목표: 버티는 역할은 유지하되 방어 신뢰도 하락
- 효과
  - 보호 효과 수치 하락

### 힐
- 목표: 지원 기능은 남기되 안정성 감소
- 효과
  - 광역 회복보다 단일 회복 우선
  
## MVP 권장 수치
- 일반 피격: 받은 피해의 `25%` 만큼 스트레스 상승
- 치명타 피격: 받은 피해의 `40%`
- 동료 사망: 전원 `+20`
- 교회 무료 관리: `-25`
- 화톳불/휴식: `-15`

## 밸런싱 메모
- 초반 MVP는 구간 효과를 단순화해서 구현하고, 패닉 세부 분기는 2종만 먼저 만든다.
- 스트레스는 체력보다 늦게 터지되, 무시하면 전투 계획이 흔들리는 수준이 적당하다.
- 딜러/탱커/서포터별 패닉 디버프는 숫자보다 행동 우선순위 변화가 더 잘 보이게 설계하는 편이 좋다.
