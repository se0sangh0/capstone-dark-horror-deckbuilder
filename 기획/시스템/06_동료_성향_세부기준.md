# 동료 성향 세부 기준

> 구현 파일: [`Assets/Scripts/Card/CardAffinity.cs`](../../Assets/Scripts/Card/CardAffinity.cs)

---

## 성향이란?

동료마다 고정된 **성향(Affinity)** 이 있으며,
성향은 해당 동료 카드의 **스택 기여량 생성 범위**를 결정한다.

카드 사용 전까지 수치는 숨겨져 있으며, 사용 시 성향 규칙에 따라 즉시 확정된다.

---

## 성향 4종

> **핵심 원칙**: 모든 성향의 기대값은 **≈+0.6**으로 수렴한다.
> 차별점은 **기대값이 아니라 분산(리스크)** 에 있다.
> 단, 도박사는 고위험·고수익 포지션으로 기대값 0을 유지한다.
> **모든 범위는 0을 제외**한다. (성향 사용 시 항상 ± 변화 발생)

| 성향 | enum 값 | 범위 (0제외) | 생성 방식 | 기대값 | 특징 |
|---|---|---|---|---|---|
| 도박사 | `Gambler` | -5 또는 +5 | 극단값 50:50 | 0.0 | 큰 위험·큰 보상, 기대값 0 |
| 안전주의자 | `Safety` | -1, 1, 2 | 균등 확률(3개) | ≈+0.67 | 작은 변동, 가장 안정적 |
| 기회주의자 | `Opportunist` | -2, -1, 1, 2, 3 | 균등 확률(5개) | +0.60 | 중간 변동, 균형 |
| 낙천가 | `Optimist` | -3, -2, -1, 1, 2, 3, 4 | 균등 확률(7개) | ≈+0.57 | 큰 변동, 완전 랜덤 |

---

## 스택 기여량 생성 규칙

- **독립시행 균등확률** 원칙 (카드_설계_프레임.md 2026-03-09 확정)
- 카드를 들고 있는 시점에는 값이 **숨겨져 있다**
- 사용 시 `AffinityHelper.GenerateStack(affinity)` 호출로 즉시 확정
- 도박사만 예외: 균등 범위가 아닌 **극단값 2개 중 하나** 선택

```csharp
// 사용 예시
int stack = AffinityHelper.GenerateStack(companion.affinity);
// companion.affinity == Gambler    → -5 또는 +5
// companion.affinity == Safety     → -1, 1, 2 중 하나
// companion.affinity == Opportunist → -2, -1, 1, 2, 3 중 하나
// companion.affinity == Optimist   → -3, -2, -1, 1, 2, 3, 4 중 하나
```

## 기대값 비교

| 성향    | 최솟값 | 최댓값 | 항목 수 | 기대값  | 분산 |
| ----- | --- | --- | --- | ---- | --- |
| 도박사   | -5  | +5  | 2개  | 0.0  | 극대 |
| 안전주의자 | -1  | +2  | 3개  | ≈+0.67 | 작음 |
| 기회주의자 | -2  | +3  | 5개  | +0.60 | 중간 |
| 낙천가   | -3  | +4  | 7개  | ≈+0.57 | 큼 |

> 모든 성향의 기대값이 ≈+0.6 으로 수렴한다.
> 선택의 기준은 기대값이 아니라 **분산(변동성)** 이다.
> 안전주의자는 작고 확실한 변화, 낙천가는 크고 예측 불가능한 변화.
> 도박사만 기대값 0으로 유지, 극단값 50:50의 고위험·고수익 포지션.
