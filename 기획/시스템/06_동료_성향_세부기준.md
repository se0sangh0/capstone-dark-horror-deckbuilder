# 동료 성향 세부 기준

> 구현 파일: [`Assets/Scripts/Card/CardAffinity.cs`](../../Assets/Scripts/Card/CardAffinity.cs)

---

## 성향이란?

동료마다 고정된 **성향(Affinity)** 이 있으며,
성향은 해당 동료 카드의 **스택 기여량 생성 범위**를 결정한다.

카드 사용 전까지 수치는 숨겨져 있으며, 사용 시 성향 규칙에 따라 즉시 확정된다.

---

## 성향 4종

| 성향 | enum 값 | 범위 | 생성 방식 | 특징 |
|---|---|---|---|---|
| 도박사 | `Gambler` | -5 또는 +5 | 극단값 50:50 | 큰 위험·큰 보상 |
| 안전주의자 | `Safety` | -1 ~ +3 | 균등 확률 | 손실 최소화, 안정적 양수 기대값 |
| 기회주의자 | `Opportunist` | -3 ~ +4 | 균등 확률 | 리스크·보상 균형, 가장 넓은 중간 범위 |
| 낙천가 | `Optimist` | -5 ~ +5 | 균등 확률 | 모든 가능성 개방, 완전 랜덤 |

---

## 스택 기여량 생성 규칙

- **독립시행 균등확률** 원칙 (카드_설계_프레임.md 2026-03-09 확정)
- 카드를 들고 있는 시점에는 값이 **숨겨져 있다**
- 사용 시 `AffinityHelper.GenerateStack(affinity)` 호출로 즉시 확정
- 도박사만 예외: 균등 범위가 아닌 **극단값 2개 중 하나** 선택

```csharp
// 사용 예시
int stack = AffinityHelper.GenerateStack(companion.affinity);
// companion.affinity == Gambler → -5 또는 +5
// companion.affinity == Safety  → -1, 0, 1, 2, 3 중 하나
```

---

## 기대값 비교

| 성향 | 최솟값 | 최댓값 | 기대값 |
|---|---|---|---|
| 도박사 | -5 | +5 | 0.0 |
| 안전주의자 | -1 | +3 | +1.0 |
| 기회주의자 | -3 | +4 | +0.5 |
| 낙천가 | -5 | +5 | 0.0 |

> 안전주의자의 기대값(+1.0)이 가장 높아 장기전에 유리하다.
> 도박사·낙천가는 기대값은 같지만 분산이 크게 다르다. (도박사는 극단 2개, 낙천가는 11개 균등)
