## 1. 개요

튜토리얼은 신규 플레이어가 전투의 기본 흐름(드로우 → 카드 사용 → 스킬 자동 발동 → 적 행동 → 미행동 보상)을 한 번 경험해보고 본 게임에 진입할 수 있도록 하는 가이드 노드다.

MVP에서는 텍스트·하이라이트 기반의 가벼운 일회성 가이드로 구현하며, 복잡한 분기·연출은 배제한다.

---

## 2. 트리거 / 진입 조건

### 2-1. 자동 실행 (최초 실행)
- 게임 실행 시 `PlayerPrefs` 의 튜토리얼 완료 플래그를 확인한다.
- **플래그가 없거나 false 인 경우 (최초 실행)** → 메인 메뉴 → 튜토리얼 자동 시작.
- 튜토리얼 진입 시 우측 상단에 **[튜토리얼 스킵] 버튼**을 항상 노출한다.
  - 스킵 시 즉시 메인 메뉴로 복귀하고 완료 플래그를 true 로 설정한다.

### 2-2. 수동 재진입 (이미 완료한 경우)
- 플래그가 이미 true 인 경우 → 메인 메뉴의 **[시작하기] 버튼 옆에 [처음이신가요?] UI** 를 노출한다.
- 클릭 시 튜토리얼 노드로 재진입할 수 있다. (완료 플래그는 그대로 유지)

### 2-3. PlayerPrefs 키 (제안)
- 키 이름: `tutorial_completed`
- 값: `0` (미완료) / `1` (완료)
- 튜토리얼 완료(스킵 포함) 시 `1` 로 설정.

## 3. 튜토리얼 시나리오

### 3-1. 파티 구성 (고정)
| 슬롯 | 직업 | 역할 |
|---|---|---|
| 0 | 캐스터 | Dealer |
| 1 | 프리스트 | Support |
| 2 | 디펜더 | Tank |

> 3인 파티로 구성. 4번째 슬롯은 비워둔다.

### 3-2. 적 구성 (고정)
- **고블린 1마리** 만 등장.
- 학습용으로 HP/공격력은 본 게임 기본 고블린 수치를 그대로 사용하거나 살짝 낮춰 학습이 빠르게 끝나도록 보정한다 (별도 데이터 조정 가능).

### 3-3. 학습 항목 (피드백 단계)
튜토리얼 진행 중 다음 항목을 순서대로 학습할 수 있도록 가이드 메시지·하이라이트를 제공한다.

1. **카드 드로우**
   - 턴 시작 시 손패가 채워지는 것을 시각적으로 강조.
   - 메시지 예시: *"턴이 시작되면 동료 수만큼 카드가 손패에 들어옵니다."*

2. **스킬 자동 사용**
   - 카드를 클릭해 스택에 반영 → 턴 종료 시 동료들이 자동으로 스킬을 발동한다는 흐름을 안내.
   - 메시지 예시: *"카드는 스택에 영향을 줍니다. 동료는 자신의 역할 스택을 기반으로 발동 가능한 가장 높은 코스트 스킬을 자동으로 사용합니다."*

3. **적 스킬 타격**
   - 적(고블린) 의 행동 → 아군 피격 시각/사운드 피드백 강조.
   - 메시지 예시: *"적도 자신의 차례에 행동합니다. 피격 시 HP / 스트레스 변화를 확인하세요."*

4. **미행동 보상**
   - 일부러 스택을 사용 안 한 상태로 턴 종료 → 미행동 시 다음 턴 **스택 +1 / 행동 순서 우선** 보상 안내.
   - 메시지 예시: *"이번 턴에 행동하지 않은 동료는 다음 턴에 스택 +1 보상과 행동 순서 우선권을 얻습니다. 단, 먼저 피격되는 리스크도 같이 있습니다."*

> 모든 피드백은 하이라이트(외곽선 강조) + 짧은 텍스트 한 줄로 처리하며, 사용자가 [확인] 또는 [다음] 버튼으로 진행한다.

### 3-4. 종료 조건
- 고블린 처치 → 승리 → 튜토리얼 완료 메시지 → `PlayerPrefs` 완료 플래그 = `1` → 메인 메뉴 복귀.
- 또는 [튜토리얼 스킵] 버튼 클릭 → 동일하게 플래그 = `1` 설정 후 메인 메뉴 복귀.

## 4. UI 구성

### 4-1. 메인 메뉴
- **시작하기** (기존)
- **[처음이신가요?]** (신규, `tutorial_completed=1` 인 경우에만 노출) — 클릭 시 튜토리얼 노드로 진입.

### 4-2. 튜토리얼 전투 화면
- 기본 전투 화면과 동일한 레이아웃.
- **우측 상단: [튜토리얼 스킵] 버튼** 상시 표시.
- **하단 중앙(또는 화면 중앙): 가이드 메시지 패널** — 단계별 메시지 + [다음] 버튼.
- **하이라이트 오버레이**: 각 단계에서 강조할 UI 요소(손패, 카드, 적, 스킬 발동 라인 등) 위에 외곽선·반투명 마스크.

## 5. 예외 처리

### 5-1. 스킵
- 어느 단계에서든 스킵 가능.
- 스킵 시 진행 중 단계 무시하고 즉시 종료 + 플래그 저장.

### 5-2. 튜토리얼 중 사망
- 일반 전투처럼 사망 처리하되, 튜토리얼은 패배 시에도 자동으로 처음부터 재시작한다 (메인 메뉴 복귀 X).
- 단 [튜토리얼 스킵] 으로는 언제든 빠져나갈 수 있다.

### 5-3. 데이터 충돌
- 튜토리얼 시작 시 기존 런 데이터(파티/덱)는 임시 격리하고 종료 시 복원한다.
- 또는 별도 세이브 슬롯에서 진행 후 결과를 본 게임에 반영하지 않는다.

- [ ] 우측 상단 스킵 버튼이 항상 노출되는가?
- [ ] 스킵 시 `tutorial_completed=1` 로 저장되는가?
- [ ] 재실행 시 자동 진입이 발생하지 않는가?
- [ ] 메인 메뉴에 [처음이신가요?] 가 노출되는가?
- [ ] [처음이신가요?] 클릭 시 튜토리얼 재진입이 동작하는가?
- [ ] 3인 파티(캐스터/프리스트/디펜더)와 고블린 1마리가 정상 생성되는가?
- [ ] 4단계 학습 피드백(드로우 / 스킬 자동 / 적 타격 / 미행동 보상) 이 순서대로 표시되는가?
- [ ] 튜토리얼 패배 시 자동 재시작 또는 종료 흐름이 정상 동작하는가?
- [ ] 튜토리얼 종료 후 본 게임 데이터에 영향이 없는가?
