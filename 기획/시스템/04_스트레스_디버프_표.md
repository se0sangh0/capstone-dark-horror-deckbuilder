# 스트레스 디버프 표

---

## MVP 구현 원칙
- 초기 MVP는 구간 효과를 단순화하고, 패닉 분기는 2종(공포 경직/과호흡)만 우선 구현한다.

---

## 기본 범위
- 스트레스 범위: `0 ~ 100`

### 상승 조건
| 원인 | 증가량 |
|---|---|
| 피격 시 | `(데미지 × 0.25) - stressResist` (최소 0) |
| 치명타 피격 시 | `(데미지 × 0.40) - stressResist` (최소 0) |
| 동료 사망 시 | 전원 +20 |

> `stressResist` 는 동료별 스트레스 저항력 수치이다.
> 예: 데미지 50 피격 시, stressResist 10 → `(50 × 0.25) - 10` = +2.5
> 예: 데미지 20 피격 시, stressResist 5 → `(20 × 0.25) - 5` = 0 (최소 0 적용)

### 기본 회복
| 방법 | 회복량 |
|---|---|
| 교회 노드 무료 관리 | -25 |
| 화툿불/휴식 노드 | -15 |
| 전투 승리 | -10 |

> 추후 카드/패시브/이벤트 노드 회복 수단 확장 예정.

## 구간별 상태

### 0 ~ 50: 안정
- 공통 효과 없음
- UI는 정상 상태로만 표기

### 51 ~ 99: 압박
- 피격 시 추가 스트레스 증가량 **`+10%`** (이미 압박 구간일 때 받는 스트레스를 ×1.10)
- 스킬 퍼포먼스 감소 **`-10%`** (해당 동료가 압박 구간이면 스킬 power ×0.90)

> ✅ **수치 확정 (2026-06-02)** — 기준 권장범위 `-5% ~ -15%` 의 중앙값 **-10%** 로 확정.
> 둘 다 인스펙터에서 조정 가능: `BattleManager.pressureSkillPenaltyPercent`(스킬 -%), `BattleManager.pressureStressGainPercent`(피격 스트레스 +%). 기본 10.
> 전투 테스트 후 체감이 약/강하면 두 값으로 미세 조정.

### 100: 패닉
- **즉시 1회 패닉 판정** 진입
- 패닉 결과
  - 공포 경직: 해당 턴 행동 불가
  - 과호흡: 스택 요구량 `+1`
- 패닉 이후 스트레스는 `50`으로 감소
- **패닉은 100을 찍을 때마다 매번 재발동**한다. (50으로 내려가도 다시 100 도달 시 패닉 발생)
- 패닉 이후의 **역할별 중증 디버프 유지안**은 백로그 항목이며, MVP에서는 적용하지 않는다.

## 역할별 중증 디버프

> ⚠️ **백로그** — MVP 범위 외. 추후 구현 예정.

### 딜러
- 목표: 화력은 유지하되 불안정성 증가
- 효과: 받는 피해량 증가

### 탱커
- 목표: 버티는 역할은 유지하되 방어 신뢰도 하락
- 효과: 보호 효과 수치 하락

### 서포터
- 목표: 지원 기능은 남기되 안정성 감소
- 효과: 광역 회복보다 단일 회복 우선

---

## 범위별 요약

| 구간 | 상태 | MVP 구현 |
| ---------- | ------------------ | -------- |
| 0~50 | 안정 | ✅ |
| 51~99 | 압박 | ✅ (스킬 -10% / 피격 스트레스 +10%) |
| 100 | 패닉 (즉시 발동, 재발동 가능) | ✅ |
| 역할별 중증 디버프 | 패닉 이후 전투 종료까지 유지 | ❌ 백로그 |

---

## 밸런싱 메모
- 스트레스는 체력보다 늦게 터지되, 무시하면 전투 계획이 흔들리는 수준이 적당하다.
- 딜러/탱커/서포터별 패닉 디버프는 숫자보다 행동 우선순위 변화가 더 잘 보이게 설계하는 편이 좋다.
