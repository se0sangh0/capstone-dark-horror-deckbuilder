# 직업군 스킬 테이블

> 구현용 데이터 스키마(스크립터블오브젝트)는 여기로: [데이터_테이블_v0](01_데이터_테이블_v0.md)

## 현재 상태
- 이 문서는 팀 내 합의 전까지 템플릿 수준으로 유지한다.
- 실제 직업명, 스킬 문안, 수치는 협의가 끝난 뒤 채운다.

## 설계 원칙(MVP)
- MVP 동료 수는 **6종**
- 역할군은 **딜러 2 / 탱커 2 / 서포터 2** 구조
- MVP에서는 각 동료가 **스킬 2개 고정**(스킬 랜덤/중복방지 등은 제외)
- 자동행동 상세 우선순위 규칙은 `전투_시스템_명세` 기준을 따른다(이 문서는 스킬 데이터 표 역할).

> 직업군/패시브/확장 시너지 설계는 백로그로 분리
