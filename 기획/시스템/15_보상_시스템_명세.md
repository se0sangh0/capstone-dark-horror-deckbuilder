# 보상 시스템 명세

> ⚠️ 영혼석 드롭 수치는 임시값. 적 수 확정 및 밸런스 테스트 후 조정 예정.

---

## 확정 구조 (2026-05-26)

| 항목 | 내용 |
|---|---|
| 보상 종류 | 영혼석만 (카드 추가 없음) |
| 지급 방식 | 적 처치 시 바닥에 오브젝트 드롭 → 자동 수거 (연출용) |
| 지급 시점 | 처치 즉시 |
| 카드 추가 보상 | ❌ 없음 |

---

## 시작 영혼석

| 항목 | 값 |
|---|---|
| 런 시작 시 보유 영혼석 | **20개** |

> 플레이어는 런을 시작할 때 영혼석 20개를 보유한 상태에서 시작한다.  
> 이후 전투 노드에서 적 처치 시 추가 획득한다.

---

## 영혼석 드롭량 (임시값 / 적 3마리 기준)

| 적 | 드롭량 | 비고 |
|---|---|---|
| 고블린 | 8개 | 일반 적 (약) |
| 약탈자 | 12개 | 일반 적 (강) |
| 거두는 자 | 20개 | 보스 고정 |

### 전투 1판 기대 획득량 (3마리 기준)

| 구성 예시 | 획득량 |
|---|---|
| 고블린 3마리 | 24개 |
| 고블린 2 + 약탈자 1 | 28개 |
| 약탈자 3마리 | 36개 |

---

## MVP 런 1회 수지 시뮬레이션

**시퀀스**: 시작 → 전투1 → 용병소 → 전투2 → 화툿불 → 보스

| 구간 | 획득 / 지출 | 누적 (최솟값 기준) |
|---|---|---|
| 런 시작 | +20개 (기본 지급) | 20개 |
| 전투1 클리어 | +24~36개 | 44~56개 |
| 용병소 모집 1회 | -30개 | 14~26개 |
| 용병소 모집 2회 | -30개 | -16~-4개 (2회는 전투 구성에 따라 불가) |
| 전투2 클리어 | +24~36개 | 회복 |
| 보스 클리어 | +20개 | +20개 |

> 💡 런 시작 영혼석(20개) 덕분에 전투1 이전 용병소가 배치되더라도 최소 1회 모집이 가능하다.  
> 전투1 후 용병소 진입 시 **44~56개 보유 → 모집 1~2회 가능** (리롤 최소화 기준).

## 연출 명세 (영혼석 드롭 오브젝트)

- 적 처치 시 해당 위치에 영혼석 오브젝트 스폰
- 스폰 후 **약 0.5초 뒤 자동 수거 애니메이션** 재생 (UI 방향으로 빨려들어가는 연출)
- 플레이어 인터랙션 없음 (자동 처리)
- Unity **Object Pooling** 적용 권장

## 백로그

| 항목 | 내용 |
|---|---|
| 영혼석 → 마석 치환 | 런 종료 후 획득한 영혼석이 마석으로 치환 (메타 성장 재화) |
| 엘리트 전투 추가 보상 | 엘리트 노드 구현 시 일반 전투보다 높은 드롭량 책정 필요 |
| 적 종류 추가 시 | 신규 적 추가될 때마다 드롭량 별도 책정 |
| 마석 치환 비율 | 영혼석 n개 → 마석 1개 (수치 미결) |
