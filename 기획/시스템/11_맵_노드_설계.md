# 맵 노드 설계

## 확정 구조 (2026-04-10 확정 / 2026-04-15 층당 선택 구조 명확화)

| 항목     | 값      | 비고             |
| ------ | ------ | -------------- |
| 총 층 수  | 10층    | 전투 테스트 후 조정 가능 |
| 층당 갈래  | 3갈래 중 1개 선택 | MVP 고정, 분기 구조 |
| 총 방문 노드 | 10개 | 층당 1개 × 10층 |
| 시작 노드  | 1층 고정  |                |
| 화툿불 노드 | 9층 고정  |                |
| 보스 노드  | 10층 고정 |                |

---

## 노드 분류

### 오픈 노드 (진입 전 내용 공개)
| 노드 | MVP 포함 | 역할 |
|---|---|---|
| 전투 노드 | ✅ 우선 구현 | 일반 적 전투, 코어 루프 증명 |
| 화툿불 노드 | ✅ 우선 구현 | 체력/스트레스 회복, 정비 |
| 보스 노드 | ✅ 우선 구현 | 10층 고정, 런 종료 조건 |

### 이벤트 노드 (진입 전 내용 비공개)
| 노드 | MVP 포함 | 역할 |
|---|---|---|
| 용병소 | ✅ 우선 구현 | 동료 모집, 영혼석 소비 |
| 교회 | ⏳ 백로그 우선순위 | 체력 회복, 부활 |
| 엘리트 전투 | ⏳ 백로그 우선순위 | 고난이도 전투, 추가 보상 |

---

## 노드 타입 비율 (초안)

| 타입  | 비율     | 비고            |
| --- | ------ | ------------- |
| 전투  | 70%    |               |
| 이벤트 | 30%    | 용병소/교회/엘리트 랜덤 |
| 보스  | 10층 고정 |               |

> ⚠️ 비율은 밸런스 테스트 후 조정 예정.

---

## 자동 생성 알고리즘 방침

- 각 층은 **3개의 노드 분기 중 1개 선택** 구조
- 층별 노드 배열은 **사전 정의 가중치 기반 랜덤** 생성
- 연속 같은 타입 3회 이상 방지 (재시도 로직)
- 1, 9, 10층은 항상 노드 고정
- 세이브 포인트: 5층 / 10층 진입 시 자동 저장

---

## 미결 수치

> ⚠️ **추후 확정 예정**
- 층당 분기 수 (현재 3갈래 고정, 가중치 분기 추가 여부 미결)
- 이벤트 노드 내 각 타입 등장 비율
- 화툿불 회복량 수치
- 교회/엘리트 구현 시점 및 수치 (백로그)
