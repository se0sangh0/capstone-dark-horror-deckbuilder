# 맵 노드 설계

## 확정 구조 (2026-04-08 확정)

| 항목 | 값 | 비고 |
|---|---|---|
| 총 층 수 | 10층 | 전속 테스트 후 조정 가능 |
| 층당 갈래 | 3갈래 | MVP 고정 |
| 노드 타입 | 전투 / 휴식 / 이벤트 / 상점 / 보스 | |
| 보스 노드 위치 | 10층 고정 | |

## 노드 타입 비율 (초안)

| 타입 | 비율 | 비고 |
|---|---|---|
| 전투 | 50% | |
| 휴식 | 20% | 스트레스 회복 |
| 이벤트 | 20% | 텍스트 기반 선택지 |
| 상점 | 10% | 마석 소비 |

> ⚠️ 비율은 밸런스 테스트 후 조정 예정.

## 자동 생성 알고리즘 방침

- 층별 노드는 **사전 정의 가중치 기반 랜덤** 선택
- 연속 같은 타입 3회 이상 방지 (재시도 로직)
- 10층은 항상 보스 노드 고정
- 세이브 포인트: 5층 / 10층 진입 시 자동 저장

## 미결 수치

> ⚠️ **추후 확정 예정**
- 층당 노드 수 (현재 3갈래 고정, 가중치 분기 추가 여부 미결)
- 이벤트 노드 분기 수 및 선택지 효과 범위
- 상점 재고 구성 및 가격 기준
- 휴식 노드 스트레스 회복량 연동 수치
