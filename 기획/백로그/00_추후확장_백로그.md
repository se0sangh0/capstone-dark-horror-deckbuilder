## 1) 코어루프 확장(프로토타입 이후)
- 전투/노드 루프에 **패시브/덱/동료 구성 조정** 요소를 본격적으로 추가
- 성향/복잡한 시너지 시스템 확장(세부 규칙은 별도 문서로 분리 예정)

## 2) 로그북/단서 수집 UI(정식 버전 후보)
- MVP에서는: 노드/환경 오브젝트에서 **텍스트 단서(오브젝트 텍스트)** 를 팝업/기록 형태로만 제공
- 정식 버전 후보:
  - 로그북/기록 조각 수집 UI
  - 수집 → 조합 → 추론 구조
  - 단계형 메타 지표(시스템 경고/사념 단계 등)

## 3) 메타 재화(마석) 확장
- 마석(마을 메타 재화) 사용처 후보:
  - 패시브 해금(우선 후보)
  - 꾸밈/커스터마이징(장기 후보)

## 4) 플레이 로그 기반 서사 시스템 ⭐ 신규
- 전투 결과(승패)가 아닌 **"어떻게 생존했는가"** 의 기록을 축적
- 각 플레이의 선택(어떤 동료를 잃었는지, 어떤 노드를 피했는지, 스트레스 한계치에 도달한 경우)을 **로그북/기록 조각** 으로 저장
- 게임 종료 후 로그를 기반으로 **나만의 생존기** 가 완성됨
- **다회차 플레이 유도** — "모든 사람이 사회를 타파하는 영웅이 되는 건 아니다. 누군가는 그저 살아남기 위해 발버둥칠 뿐이다" 라는 테마 전달

## 5) 스트레스 시스템 서사 확장 (정식 버전 후보) ⭐ 신규
- (MVP) 수치적 패널티로 전투 리스크 조절 (받는 피해량 증가 등)
- (정식 버전) 동료가 '인격 조각'으로서 **괴이(怪異) 현상** 을 일으키는 서사 장치로 확장
  - 환각: 아군 오인 공격, 허상 타격
  - 도주: 전투 중 이탈
  - 배신: 아군 대상 스킬 발동
  - 동료가 개개인의 캐릭터가 아닌 현상의 일부라는 설정과 결합

## 6) 성향 기반 전략 다변화 ⭐ 신규
- 동료의 '성향' 조합에 따라 **디메리트 발동 → 이로운 효과** 로 전환하는 시너지
- 예: 신중 + 과감한 조합 → 특정 조건에서 보너스 스택 생성
- 수집 시 성향 조합 전략으로 덱 빌딩과 전투 다변화
- 모집 단계에서 성향을 고려한 전략적 편성 유도

## 결정 필요(체크)
- [ ] MVP에서 오브젝트 텍스트는 **"팝업만"** 인지, **"기록(간단 로그)"** 까지 포함할지
- [ ] 마석을 MVP에서 아예 숨길지(용어/아이콘 노출 X), 아니면 "미사용 재화"로만 노출할지
- [ ] 플레이 로그 기반 서사의 MVP 반영 범위: MVP에서는 단순 기록만? → 조합/추론은 정식 버전?
