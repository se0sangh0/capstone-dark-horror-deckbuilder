# 패시브 초안

## 문서 성격
- 이 문서는 **패시브 아이디어를 빠르게 모아두는 초안 메모**다.
- 아직 확정 명세가 아니며, 실제 반영 전에는 밸런스/우선순위 검토가 필요하다.
- 정리된 백로그 문서는 [시스템/13_패시브_스킬_및_추후개발_백로그.md](../시스템/13_패시브_스킬_및_추후개발_백로그.md)를 참고한다.

## 활용 방식
- 여기에는 아이디어를 자유롭게 추가한다.
- 실제 구현 후보로 올릴 항목은 정리 후 시스템 문서로 이동한다.
- MVP 포함 여부가 불명확한 항목은 초안 상태로 유지한다.

## 1. 전투 보상 관련
- 전투 후 획득 영혼석 N% 증가
- 전투 후 체력 회복 N% 증가
- 전투 후 스트레스 회복 N% 증가

## 2. 전투 내 관련
- 전투 시작 시 체력 N% 증가
- 전투 내 아군이 주는 피해량 N% 증가
- 전투 내 받는 피해량 N% 감소

## 3. 모집 관련
- 특정 역할군 모집 확률 상승
- 특정 성향 모집 확률 상승

## 4. 탐사 관련
### 런 방향성 선택 패시브
- 이 계열은 이번 런의 운영 방향을 미리 정하게 만드는 메타 선택지다.
- 패시브 선택은 게임 시작 시 메인 화면에서 먼저 진행하거나, 이전 런이 모두 종료된 뒤 다음 런 준비 단계에서 진행한다.
- 목적은 특정 콘텐츠(전투/용병소/이벤트) 조우 확률을 조정해 덱 성장 방향과 리스크 구조를 바꾸는 것이다.

#### 선택 규칙
- 동일 카테고리 내 효과는 상호 배타적으로 취급한다.
- 이미 선택한 패시브와 상충하는 항목은 UI에서 선택할 수 없도록 비활성화한다.
- 한 번 선택한 런 방향성 패시브는 해당 런이 끝날 때까지 유지한다.

#### 선택지 예시
- 전투 추적: 이벤트 노드에서 전투 발생 확률 증가
- 회피 경로: 이벤트 노드에서 전투 발생 확률 감소
- 인력 수급: 이벤트 노드에서 용병소 확률 증가
- 독자 생존: 이벤트 노드에서 용병소 확률 감소

## 5. 동료 관련
### 캐스터 3성 달성 시 해금
- 매직 미사일 타겟 증가
- 파이어볼 데미지 증가
- 매직 미사일로 적 처치 시 연속 발동

### 오펜더 3성 달성 시 해금
- 발도 데미지 증가
- 발도 연속 발동 시 콤보 적용
  - 콤보당 추가 데미지 10% 증가
  - 해당 오펜더가 다른 스킬 사용 시 콤보 리셋
- 일섬 데미지 증가

### 디펜더 3성 달성 시 해금
- 배치 순서 최전열 고정
- 방어 준비 실드량 증가
- 전투 태세 본인 대상 받는 피해량 감소 추가
- 전투 태세 링크 효과 추가
  - 모든 아군이 받는 피해의 30%씩 대신 나눠받음

### 어택커 3성 달성 시 해금
- 무모한 강타 데미지 증가
- 불굴 체력 회복량 증가
- 불굴 발동 시 링크 효과 추가
  - 모든 아군이 받는 피해의 10%씩 대신 나눠받음
- 무모한 강타 본인 대상 받는 피해량 감소 추가
- 무모한 강타 발동 시 최전열 이동

### 프리스트 3성 달성 시 해금
- 별부름 체력 회복량 증가
- 기원 체력 회복량 증가
- 기원 발동 시 본인 대상 받는 피해량 감소
