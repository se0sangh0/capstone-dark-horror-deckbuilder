# MVP 범위 정의 (2026-03-15 확정 / 2026-04-10 노드 구조 갱신)

**완료 기준**: 15~20분 루프 완주, 노드 6종 구현 (우선순위 분류 있음)

## 확정 수치
| 카테고리 | 수량/범위 | 세부 |
|---------|-----------|------|
| 카드 | 3종류 | 딜러·탱커·서포터 스택 생성용 각 1종류 (-5 ~ +5) |
| 동료 | 6종 | 파티 4인 고정 (모집 교체) |
| 적 | 6종 | 노말 3 / 엘리트 2 / 보스 1 |
| 노드 | 6종 | 오픈 노드 3종 + 이벤트 노드 3종 (아래 표 참조) |

> **카드 구조 메모**: 카드는 스택 타입(딜러/탱커/서포터)별 1종씩 총 3종류.
> 각 카드는 성향에 따라 -5 ~ +5 범위 내에서 스택을 생성한다.
> 카드 종류 확장(타입별 복수 카드)은 백로그(`백로그/00_추후확장_백로그.md`)로 이관.

## 노드 구조

### 오픈 노드 (진입 전 내용 공개)
| 노드 | 우선순위 | 역할 |
|---|---|---|
| 전투 노드 | ✅ 우선 구현 | 일반 적 전투, 코어 루프 증명 |
| 화툿불 노드 | ✅ 우선 구현 | 체력/스트레스 회복, 정비 |
| 보스 노드 | ✅ 우선 구현 | 10층 고정, 런 종료 조건 |

### 이벤트 노드 (진입 전 내용 비공개)
| 노드 | 우선순위 | 역할 |
|---|---|---|
| 용병소 | ✅ 우선 구현 | 동료 모집, 영혼석 소비 |
| 교회 | ⏳ 백로그 우선순위 | 체력 회복, 부활 |
| 엘리트 전투 | ⏳ 백로그 우선순위 | 고난이도 전투, 추가 보상 |

## 포함 vs 제외
| 포함 (MVP 필수) | 제외 (백로그) |
|--------------------|----------------|
| 맵 노드 이동 | 동료 승급 시스템 |
| 공용 스택 생성/소모 | 전체 히든 노드 |
| 자동 전투 (스킵/미행동) | 스트레스 밸런싱 |
| 스트레스 기본 | 성향/복수 시너지 |
| 전투 UI 골격 | 발표 폴리싱 |
| 교회 노드 (구현, 후순위) | |
| 엘리트 전투 노드 (구현, 후순위) | |
