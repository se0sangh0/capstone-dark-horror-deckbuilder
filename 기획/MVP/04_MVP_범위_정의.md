# MVP 범위 정의 (2026-03-15 확정)

**완료 기준**: 15~20분 루프 완주, 노드 4종으로 탐험→전투→보스 흐름 성립

## 확정 수치
| 카테고리 | 수량/범위 | 세부 |
|---------|-----------|------|
| 카드 | 3종 | -5 ~ +5 |
| 동료 | 6종 | 파티 4인 고정 (모집 교체) |
| 적 | 6종 | 노말 3 / 엘리트 2 / 보스 1 |
| 노드 | 4종 | 전투/용병소/교회/보스 |

## 포함 vs 제외
| 포함 (MVP 필수) | 제외 (백로그) |
|--------------------|----------------|
| 맵 노드 이동 | 동료 승급 시스템 |
| 공용 스택 생성/소모 | 전체 히든 노드 |
| 자동 전투 (스킵/미행동) | 스트레스 밸런싱 |
| 스트레스 기본 | 성향/복수 시너지 |
| 전투 UI 골격 | 발표 폴리싱 |
