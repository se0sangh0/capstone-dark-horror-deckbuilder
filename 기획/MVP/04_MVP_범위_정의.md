# MVP 범위 정의

## 포함
- 맵 노드 이동
- 카드로 공용 스택 생성
- 손패 매 턴 리셋
- 고정 배치 동료 자동행동
- 스택 부족 시 행동 스킵
- 미행동 보상(스택 +1 / 다음 턴 우선순위 상승)
- 스트레스 누적 및 회복 노드 처리
- 전투 UI 초안 및 플레이어 화면 골격

## 2026-03-09 확정 수치
- 카드: 3종
- 카드 수치 범위: `-5 ~ +5`
- 동료: 6종
- 적 구성
  - 노말몹 3종
  - 엘리트 2종
  - 보스 1종
- 노드: 4종

## MVP 제외 또는 후순위
- 동료 조합을 통한 상위 동료 승급
- 전체 히든 노드 풀 구현
- 세부 스트레스 디버프 수치 밸런싱
- 복잡한 성향 시스템 전체 구현
- 복수의 시너지 조합 확장
- 발표용 연출 및 폴리싱

- 15~20분 플레이 루프 1회 완주 가능
