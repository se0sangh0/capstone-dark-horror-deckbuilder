# 직업군 스킬 테이블

## 현재 상태
- 이 문서는 팀 내 합의 전까지 템플릿 수준으로 유지한다.
- 실제 직업명, 스킬 문안, 수치는 협의가 끝난 뒤 채운다.

## 설계 원칙
- MVP 동료 수는 6종
- 역할군은 딜 2, 탱 2, 힐 2 구조
- 각 동료는 `역할군 카드 + 스킬1 + 스킬2 + 패시브` 조합으로 생성
- 자동행동 시 여러 선택지가 있으면 코스트가 높은 스킬을 우선 사용

## 직업군 예시

### 직업명
- 역할군: 탱딜힐
- 패시브 
  - N스택 + M
- 스킬 1 (소모 스택)
- 스킬 2 (소모 스택)
